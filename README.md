# JWT Algorithm Confusion Demo

Демонстрация атаки Algorithm Confusion на JWT токенах.

## Описание

Веб приложение демонстрирует уязвимость подмены алгоритма в JWT. Переключатель защиты показывает разницу между уязвимой и безопасной реализацией.

## Функционал

- Регистрация пользователей
- Вход и получение JWT токена
- Доступ к защищённому ресурсу
- Атака Algorithm Confusion
- Защита через явное указание алгоритма

## Запуск

1. Скачать файл `jwt_demo.html`
2. Открыть в любом современном браузере
3. Нажать "Быстрый старт"
4. Переключать защиту и наблюдать разницу

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Симуляция JWT
